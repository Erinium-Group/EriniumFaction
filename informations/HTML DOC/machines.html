Tourelle Automatique
Combat
📋 Description

Système de défense automatisé qui cible et tire sur les joueurs ennemis entrant dans votre territoire. Utilise un système d'identification par faction pour éviter les tirs amis.

✨ Fonctionnalités
  • Détection automatique des joueurs ennemis
  • Portée configurable selon le tier (15-50 blocs)
  • Types de munitions : flèches, potions, projectiles explosifs
  • Rotation 360° avec priorisation des cibles
  • Mode veille pour économiser l'énergie
  • Désactivable par redstone ou switch manuel
🔧 Tiers de Tourelle
Tier 1 : Portée 15 blocs, flèches normales, 1 tir/2s
Tier 2 : Portée 30 blocs, flèches + potions, 2 tirs/s
Tier 3 : Portée 50 blocs, toutes munitions, 3 tirs/s, double cible
Consommation
50-200 RF/tir
Capacité Munitions
128-512 items
Limitations

Ne peut pas tirer à travers les blocs solides. Vulnérable aux TNT et aux attaques à distance. Peut être hackée par des joueurs avec des items spéciaux.

Beacon Amélioré de Faction
Défense
📋 Description

Version améliorée du beacon vanilla qui applique des effets de buffs puissants uniquement aux membres de votre faction dans un rayon étendu. Parfait pour fortifier une zone de guerre.

✨ Fonctionnalités
  • Effets réservés aux membres de la faction
  • Portée étendue : 100-300 blocs selon tier
  • Buffs stackables avec plusieurs beacons
  • Effets custom : Résistance aux explosions, Régénération améliorée
  • Indicateur visuel de la zone d'effet (particules)
  • Activation/désactivation via GUI
🔧 Tiers de Beacon
Tier 1 : 100 blocs, 2 effets simultanés, niveau I
Tier 2 : 200 blocs, 3 effets simultanés, niveau II
Tier 3 : 300 blocs, 5 effets simultanés, niveau III
Effets Disponibles

Speed, Strength, Resistance, Regeneration, Jump Boost, Fire Resistance, Absorption, + Effets custom : Blast Resistance (réduit dégâts explosion 30%), Combat Focus (+15% dégâts critiques)

Consommation
500 RF/s
Coût Activation
Nether Star
📡
Détecteur de Mouvement
Sécurité
📋 Description

Système de surveillance avancé qui détecte tout mouvement dans votre base et envoie des alertes en temps réel sur Discord via webhook. Parfait pour surveiller votre territoire même offline.

✨ Fonctionnalités
  • Détection des joueurs ennemis uniquement
  • Envoi automatique sur Discord webhook
  • Informations : pseudo, coordonnées, timestamp, screenshot (optionnel)
  • Rayon de détection : 20-100 blocs selon tier
  • Filtres configurables (membres, alliés, tous)
  • Mode furtif (pas de particules visibles)
  • Logs internes sauvegardés
🔧 Tiers de Détecteur
Tier 1 : 20 blocs, alerte texte simple
Tier 2 : 50 blocs, embed Discord avec coordonnées
Tier 3 : 100 blocs, embed + screenshot + historique
Format Message Discord

🚨 INTRUSION DÉTECTÉE
👤 Joueur: EnemyPlayer
📍 Position: X:1234 Y:64 Z:5678
🕐 Heure: 14:32:15
🏴 Faction: EnnemiFaction

Cooldown Alerte
30s - 5s
Consommation
10 RF/s
Configuration Webhook

Configurez l'URL du webhook Discord dans le GUI de la machine. Testez la connexion avec le bouton "Test Alert". Permissions requises : Grade Officier minimum dans la faction.

Raffinerie de Minerais
Production
📋 Description

Machine de traitement avancée qui multiplie vos rendements de minerais bruts. Plus vous investissez d'énergie, plus vous obtenez de ressources en retour. Essential pour l'économie de faction.

✨ Fonctionnalités
  • Multiplication des minerais : x2 à x4 selon tier
  • Bonus aléatoire de minerais rares (5-15% de chance)
  • Processus par étapes : broyage → lavage → raffinage
  • Interface multibloc avec animations visuelles
  • Compatible avec tous les minerais (vanilla + modded)
  • Système de file d'attente automatique
🔧 Tiers de Raffinerie
Tier 1 : x2 minerais, 100 RF/opération, 10s/minerai
Tier 2 : x3 minerais, 250 RF/opération, 7s/minerai, 5% bonus rare
Tier 3 : x4 minerais, 500 RF/opération, 5s/minerai, 15% bonus rare
Bonus Rares

Lors du raffinage de fer → chance de diamant | Or → chance d'émeraude | Cuivre → chance de redstone | Diamant → chance de netherite scrap

Capacité Input
64 stacks
Capacité Output
128 stacks
Vitesse Max
12 items/s
Buffer Énergie
100k RF
Structure Multibloc

3x3x4 blocs. Base en acier renforcé, parois en verre industriel, cœur de traitement en électrum. Utilisez le Guide du Constructeur (item) pour voir la structure exacte en jeu.

🔥
Fonderie d'Alliages
Production
📋 Description

Four haute température permettant de créer des alliages avancés en combinant différents métaux. Contrôlez la température et le temps de fusion pour des résultats optimaux.

✨ Fonctionnalités
  • Création d'alliages : Bronze, Électrum, Cupronickel, etc.
  • Système de température (500°C - 2000°C)
  • Recettes avec ratios précis (90/10, 70/30, etc.)
  • Visualisation du processus de fonte (liquide animé)
  • Mode automatique avec hoppers
  • Chance de production bonus (qualité supérieure)
🔧 Recettes d'Alliages
Bronze : 90% Cuivre + 10% Étain → 800°C, 20s
Électrum : 50% Or + 50% Cuivre → 1200°C, 30s
Cupronickel : 75% Cuivre + 25% Nickel → 1400°C, 25s
Acier : 98% Fer + 2% Charbon → 1500°C, 40s
Consommation
150-400 RF/s
Capacité Creuset
16 ingots
Temps Chauffe
5-45s
Rendement
100-110%
Attention

Une température incorrecte peut ruiner le lot ! Bronze trop chaud = perte de matériaux. Surveillez la jauge de température et suivez les recettes avec précision.

Interface GUI

Slots Input (2x) pour les métaux, jauge de température avec indicateurs de zones optimales, timer de fusion, slot output pour l'alliage produit. Mode auto activable via toggle.

Assembleur Automatique
Automatisation
📋 Description

Machine d'auto-crafting intelligente qui automatise la production d'items complexes. Programmez vos recettes favorites et laissez la machine travailler en continu.

✨ Fonctionnalités
  • Auto-craft de toutes les recettes vanilla et modded
  • File d'attente de production (jusqu'à 10 recettes)
  • Sauvegarde de 20 recettes préférées
  • Importation automatique depuis coffres connectés
  • Vitesse de craft ajustable (1-10 items/s)
  • Interface utilisateur intuitive avec recherche
  • Mode bulk (craft 64x d'un coup)
🔧 Tiers d'Assembleur
Tier 1 : 1 craft/s, 5 recettes sauvegardées, 50 RF/craft
Tier 2 : 5 crafts/s, 10 recettes sauvegardées, 30 RF/craft
Tier 3 : 10 crafts/s, 20 recettes sauvegardées, 20 RF/craft, priorités
Exemples d'Utilisation

Production de masse d'armures pour faction wars | Craft automatique de potions de combat | Fabrication continue d'outils pour les membres | Assembly d'items complexes (pistons, dispensers)

Slots Input
27 (1 chest)
Slots Output
54 (2 chests)
Buffer Craft
256 items
Vitesse Max
10 items/s
Gestion des Recettes

Cliquez sur "Apprendre Recette" en tenant un item crafté pour l'enregistrer. Organisez vos recettes par catégories (Combat, Construction, Redstone). Export/Import de profils de recettes via items "Pattern Card".

Station de Réparation de Faction
Support
📋 Description

Système de réparation électrique qui répare automatiquement les items équipés et dans l'inventaire des joueurs de la faction dans un rayon configurable. Fini les items cassés en pleine bataille !

✨ Fonctionnalités
  • Réparation automatique en temps réel
  • Affecte tous les membres dans le rayon
  • Répare armures, outils, armes, et items modded
  • Bonus de durabilité (+10-30% selon tier)
  • Priorité aux items équipés puis inventaire
  • Indicateur de progression au-dessus de la tête
  • Désactivation manuelle possible par joueur
🔧 Tiers de Station
Tier 1 : 10 blocs, 1 durabilité/s, 100 RF/durabilité
Tier 2 : 25 blocs, 5 durabilité/s, 50 RF/durabilité, +10% bonus
Tier 3 : 50 blocs, 15 durabilité/s, 25 RF/durabilité, +30% bonus
Rayon Max
50 blocs
Joueurs Simultanés
10 max
Vitesse Réparation
15 dur/s
Buffer Énergie
500k RF
Effets Visuels

Particules vertes en spirale autour du joueur pendant la réparation. Barre de progression au-dessus de la tête montrant l'item en cours de réparation. Son de cliquetis métallique apaisant.

Optimisations

Mode "Urgence" : répare uniquement les items à moins de 20% de durabilité pour économiser l'énergie. Mode "Passif" : réparation lente mais continue (parfait pour les bases). Whitelist/blacklist d'items configurables.

Limitations

Ne peut pas réparer les items avec durabilité infinie. Les items maudits (curse of vanishing/binding) nécessitent 3x plus d'énergie. La réparation s'arrête si le buffer d'énergie tombe sous 10%.

Block Miné
Piège
📋 Description

Alternative discrète à la plaque de pression. Ce bloc se déclenche lorsqu'un joueur ennemi le mine, permettant des pièges créatifs et défenses surprises dans votre base.

✨ Fonctionnalités
  • Se déclenche uniquement au minage, pas au passage
  • Détection faction : ne se déclenche que pour les ennemis
  • Apparence configurable (imite d'autres blocs)
  • Actions : explosion, signal redstone, alert Discord, TP, poison
  • Durabilité configurable avant explosion (1-10 hits)
  • Mode "Réarmement auto" après 5 minutes
  • Compatible avec redstone pour chaînes de pièges
🔧 Types d'Actions
Explosion : TNT, Creeper, ou custom (force 1-4)
Effets : Poison, Blindness, Slowness, Wither (10-30s)
Redstone : Signal 1-15, durée 1-10s
Alerte : Message faction + optionnel Discord webhook
TP Piège : Téléporte l'ennemi dans une cellule
Déguisements
50+ blocs
Cooldown Réarm
5 min
Stratégies d'Utilisation

Faux Minerais : Déguisez en diamant ore dans votre mine
Murs Piégés : Intégrez dans les murs de votre base
Coffres Explosifs : Placez sous des coffres appâts
Chaîne de Pièges : Connectez plusieurs blocs en cascade

Attention

Peut affecter les membres de votre faction si mal configuré ! Toujours tester en créatif. Les explosions peuvent endommager votre propre base. Utilisez avec précaution près de vos valuables.

Configuration Avancée

Clic droit avec wrench pour ouvrir le GUI. Sélectionnez le déguisement, l'action, l'intensité. Possibilité de créer des "Profils" de pièges pour déploiement rapide. Shift+clic droit pour copier/coller la config entre blocks.
